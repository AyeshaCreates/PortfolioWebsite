## 🚀 Live Demo & Deployment
This website is built using 100% static **HTML5**, **CSS3**, and **Vanilla JavaScript**. It runs with zero dependencies, requiring no Node.js, npm, bundlers, compilers, or local dev server. 

### 1. Local Run

1. Double-click the [index.html](file:///C:/Users/amair/.gemini/antigravity/scratch/ayesha-portfolio/index.html) file or open it directly in any modern browser.

### 2. Deployment on GitHub Pages
Deploying is simple and free:
1. Initialize a Git repository in this folder:
   ```bash
   git init
   git add .
   git commit -m "feat: initial portfolio release"
   ```
2. Create a new repository on GitHub and link it as `remote`:
   ```bash
   git remote add origin https://github.com/yourusername/your-repo-name.git
   git branch -M main
   git push -u origin main
   ```
3. Go to **Settings** > **Pages** in your GitHub repository.
4. Set the Source to **Deploy from a branch**, select the `main` branch, set the folder to `/ (root)`, and click **Save**.
5. Your portfolio will be live at `https://yourusername.github.io/your-repo-name/` in less than a minute.

## 🎨 Design System & Highlights
- **Cosmic Obsidian Theme**: Obsidian-black background (`#030712`) by default, styled with high-contrast text rendering and responsive slate-white theme transitions.
- **Typography**: Geometric `Space Grotesk` headings for a technical engineering appearance and `Inter` for highly readable body copy.
- **Interactive Particles**: A custom, lightweight HTML5 `<canvas>` rendering neural network nodes and path intersections that respond gently to mouse interaction.
- **Recruiter Actions Bar**: Features a dedicated panel right at the hero containing quick links to copy the email address in one-click, download the PDF resume, and visit external social profiles.
- **Performance & SEO**: Fully semantic tags, Open Graph tags, crawler controls (`robots.txt`), and structured metadata files (`sitemap.xml`).
